2019中国大学生程序设计竞赛（CCPC）---网络选拔赛-1004-path

发表于 2018-03-24 | 更新于 2020-01-22 | 分类于 题解(XCPC) | 评论数：

考虑维护按照边权最小的堆，维护结点信息如下:

一开始，先将每个结点从最短的那条边扩展，然后对于每次操作。取队头元素，当前的路径距离就是第$idx$小的路径，用队头元素进行扩展:

  1. 从结点$v$最短的一条边扩展
  2. 从结点u的$id$下标编号进行扩展

扩展的时候维护上述信息。

这种贪心策略就是很对。
